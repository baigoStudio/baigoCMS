调用显示

接口说明

用于显示单个调用详细信息。

URL

http://www.domain.com/bg_api/api.php?mod=call

HTTP 请求方式

GET

返回格式

JSON

请求参数
名称 类型 必须 具体描述
act string true 接口调用动作，值只能为 read。
app_id int true 应用的 APP ID，后台创建应用时生成的 ID。详情查看 查看应用。
app_key string true 应用的 APP KEY，后台创建应用时生成的 KEY。详情查看 查看应用。
call_id int true 调用 ID
 

返回结果

返回结果会根据不同的调用类型有所不同，详情请查看 文章、栏目、TAG、专题、友情链接 等有关信息。关于调用以及调用 ID 请查看 调用管理。

 
top
 

友情链接

友情链接可以通过调用来显示，调用后返回的数组结构如下。

常用语言资源
键名 类型 说明
link_id int 链接 ID
link_name string 链接名称
link_url string 链接地址
link_blank bool 是否打开新窗口
 

友情链接数据示例

[
    {
        "link_id": "4",
        "link_name": "baigo",
        "link_url": "http://www.baigo.net"
        "link_blank": "1"
    }
]
