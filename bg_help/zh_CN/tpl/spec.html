专题列表

文件名：spec_list.tpl

显示网站内所有的专题。

键名 类型 说明 备注
{$tplData.specRows} array 专题列表 详情请查看 specRow。
{$tplData.search} array 搜索参数 显示专题列表所需要的搜索参数，查看 {$tplData.search}。纯静态模式下不提供
{$tplData.query} string 搜索参数序列化字符串 搜索参数序列化为字符串以后的结果。纯静态模式下不提供
{$tplData.urlRow} array 专题列表 URL 数组 spec_url 专题列表 URL，page_attach 为分页附加参数，主要用于分页，page_ext 扩展名（仅用于纯静态模式）。
{$tplData.pageRow} array 分页参数 详情请查看 分页参数
 

{$tplData.search} 数组

显示文章列表所需要的搜索参数。

键名 类型 说明 备注
key string 搜索关键词
 
top
 

专题显示

文件名：spec_show.tpl

用于显示所有隶属于此专题的文章。

键名 类型 说明 备注
{$tplData.specRow} array 当前专题详细信息 查看 {$tplData.specRow}。
{$tplData.articleRows} array 文章列表 所有隶属于此专题的文章。详情请查看 文章。
{$tplData.pageRow} array 分页参数 详情请查看 分页参数
 

{$tplData.specRow} 数组

当前专题详细信息

键名 类型 说明 备注
spec_id int 专题 ID
spec_name string 专题名称
spec_content string 专题内容
spec_status string 专题状态 pub 为发布，hide 为隐藏。
urlRow array 当前专题 URL 数组 spec_url 为当前专题 URL，page_attach 为分页附加参数，主要用于分页，page_ext 扩展名（仅用于纯静态模式）。
rcode string 返回代码 显示当前专题的状态，详情请查看 返回代码。
 

专题数据示例

{$tplData.specRow|@print_r} 

Array (
    [spec_id] => 11 //专题 ID
    [spec_name] => 专题名称 //专题名称
    [spec_status] => show //状态
    [spec_content] => 
专题内容
 //内容
    [urlRow] => Array (
        [spec_url] => /cms/spec/id-11/ //URL
        [page_attach] => page- //分页附加
        [page_ext] => .html
    )
    [rcode] => y180102 //返回代码
)
