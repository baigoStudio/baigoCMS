模板概述

从 v2.0 版本起，baigo CMS 支持直接使用 PHP 作为模板（推荐），同时继续支持 Smarty 模板，但对 Smarty 引擎不再做更新，并逐步弃用。baigo CMS 的后台已经全部弃用 Smarty 模板。

关于 Smarty 模板的详情请查看 Smarty 官方网站 http://www.smarty.net。

前台模板位于 ./bg_tpl/pub 目录下，一套模板单独一个目录，如默认模板 ./bg_tpl/pub/default，以下教程全部以此为基础。注：模板目录必须使用 英文 与 数字，不能使用中文、符号等。

模板目录结构说明
文件名 说明 备注
index.php 网站主页
error.php 提示信息 一般用于出错信息的显示。
cate_show.php 栏目主页 用于显示所有隶属于此栏目的文章，以及栏目介绍等信息。
cate_single.php 单页栏目 单页是指该栏目没有下属的文章，只显示栏目的具体内容，适合联系方式、单位介绍等，此模板与 栏目设置 有关，只有当栏目设置为单页时，系统才会调用此模板。
article_show.php 文章显示
tag_show.php TAG 显示 用于显示所有与此 TAG 关联的文章。
spec_list.php 专题列表 显示网站内所有的专题。
spec_show.php 专题显示 用于显示所有隶属于此专题的文章，以及专题介绍等信息。
search_show.php 搜索显示 用于显示搜索结果。
include 包含文件目录 此目录用来存放被包含的文件，可自定义名称，也可删除此目录，与具体的模板写法有关。
 
top
 

调用模板位于 ./bg_tpl/call 目录下，如默认模板 ./bg_tpl/call/call_cate.php。注：模板文件名必须使用 英文 与 数字，不能使用中文、符号等。

默认模板结构说明
默认模板 调用类型
call_article.php 文章列表、日排行、周排行、月排行、年排行、总排行
call_cate.php 栏目列表
call_spec.php 专题列表
call_tag.php TAG 列表、TAG 排行
call_link.php 友情链接
